Elementos de Formulario Usados

  • <form> - Contenedor del formulario
  • <input type="text"> - Campo de texto
  • <input type="email"> - Campo de email
  • <input type="password"> - Campo de contraseña
  • <input type="date"> - Selector de fecha
  • <input type="number"> - Campo numérico
  • <input type="tel"> - Campo de teléfono
  • <input type="url"> - Campo de URL
  • <input type="color"> - Selector de color
  • <input type="range"> - Control deslizante
  • <input type="checkbox"> - Casilla de verificación
  • <input type="radio"> - Botón de opción
  • <select> - Lista desplegable
  • <optgroup> - Agrupador de opciones dentro de un select
  • <textarea> - Área de texto multilínea
  • <button> - Botón
  • <fieldset> y <legend> - Agrupación de campos
  • <label> - Etiqueta para controles
